# Instituto Andrés Bello - Portal Web Estudiantil

Proyecto desarrollado para la asignatura **Taller de Integración de Software**.

## Descripción

Este proyecto corresponde al desarrollo progresivo de un portal web para el Instituto Andrés Bello, realizado como parte de las actividades prácticas de la asignatura.

Actualmente el proyecto contempla un formulario de registro estudiantil desarrollado en HTML5, una hoja de estilos CSS para mejorar su presentación y una página adicional destinada a visualizar la información de las carreras impartidas por la institución.

## Funcionalidades implementadas

- Formulario de registro de estudiantes.
- Campos para:
  - Nombre
  - Apellido
  - RUT
  - Dirección
  - Carrera a cursar
  - Fecha del período académico
- Botones Registrar y Limpiar.
- Hoja de estilos CSS externa.
- Diseño adaptable (Responsive Design).
- Página de información de carreras.
- Navegación entre páginas mediante enlaces HTML.

## Tecnologías utilizadas

- HTML5
- CSS3
